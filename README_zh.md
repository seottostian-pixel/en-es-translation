# 英语到西班牙语神经机器翻译

基于 Transformer 的序列到序列模型，使用 PaddlePaddle 实现英语 → 西班牙语翻译。

---

## 数据集

- 来源：`spa.txt`（制表符分隔的英语/西班牙语句对）
- 总句对数：约 118,964
- 划分：70% 训练集 / 15% 验证集 / 15% 测试集

---

## 数据预处理

1. **专有名词处理** — 非句首位置首字母大写、词频低（≤3）且在西班牙语译文中原样出现的词，被标记为 `<properN>` 并在翻译过程中保留。
2. **文本规范化** — 小写化、标点符号加空格（如 `word .`）、特殊字符处理（`¿`、`¡`）。
3. **词表构建** — 按词频排序构建词到 id 的映射字典，上限 30,000；实际大小：英语约 11,638，西班牙语约 22,160。
4. **序列填充/截断** — 编码器输入固定长度 20，解码器固定长度 21（含 `[start]`/`[end]` 标记）。

---

## 模型架构

单层 Transformer 编码器–解码器结构。

| 组件 | 详情 |
|---|---|
| 词嵌入 | 可学习的 token 嵌入 + 位置嵌入 |
| 嵌入维度 | 256 |
| 前馈网络维度 | 2048 |
| 注意力头数 | 8 |
| Dropout | 0.2（训练阶段） |
| 输出层 | Linear(256 → vocab_size) |

### TransformerEncoder（编码器）
- 多头自注意力（含 padding mask）
- 前馈网络：Linear(256→2048) → ReLU → Linear(2048→256)
- 两层 LayerNorm（注意力后、前馈后各一层）

### TransformerDecoder（解码器）
- 带掩码的多头自注意力（causal mask）
- 对编码器输出的交叉注意力
- 前馈网络：结构同编码器
- 三层 LayerNorm

## 训练配置

| 超参数 | 值 |
|---|---|
| 训练轮数 | 10 |
| 批大小 | 64 |
| 基础学习率 | 0.0005 |
| 学习率调度 | 线性预热（500 步）+ 余弦退火 |
| 优化器 | Adam |
| 梯度裁剪 | 1.0 |
| 损失函数 | 带掩码的交叉熵 |

训练损失从约 97 下降至约 4.2，验证损失在第 6 轮后稳定在约 13。

## 评估结果

在测试集上使用贪心解码（批大小 256）：

| 指标 | 分数 |
|---|---|
| BLEU | **43.52** |
| 1-gram 精确率 | 72.6% |
| 2-gram 精确率 | 50.0% |
| 3-gram 精确率 | 36.7% |
| 4-gram 精确率 | 27.3% |
| 简短惩罚（BP） | 0.9963 |

---

## 推理说明

- 词表中未收录的专有名词在推理时自动识别，通过 `<properN>` 标记机制原样传递到译文。
- 多句输入按 `.`、`!`、`?` 拆分后逐句翻译。
- 西班牙语成对标点（`¿`/`?`、`¡`/`!`）在输出中自动补全。
